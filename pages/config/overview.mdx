# API

Геопортал описывается в виде конфигурации в формате JSON. Конфигурация включает в себя настройку:

- источников данных
- слоёв
- визуализаций
- карточек
- фильтров
- копирайтов.

[Пример конфигурации слоёв](https://github.com/sloyio/sloy-template/tree/main/src/gorodets) (доступ по запросу)

[Результат](https://sloy.io/gorodets/)
