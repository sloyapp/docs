# Конфигурация

Геопортал описывается в виде конфигурации в формате JSON. Конфигурация включает в себя настройку:

- источников данных
- слоёв
- визуализаций
- карточек
- фильтров
- копирайтов.

В это размере приведены визуальныйе примеры. Сама конфигурация в виде кода описана в соседнем разделе [API](/tutorial/api).

## Cписок источников данных (sources)

Источниками данных являются файлы или запросы к API.

Доступные форматы файлов: geojson, json, mvt\pmtiles. Мы можем добавить другие форматы по запросу.

## Cписок слоёв (layers)

Слои отображаются в левой части странце. Например, «Количество жителей», «Здания», «Возраст зданий», «Коммерция», «Туристические маршруты» на скриншоте ниже. Все слои могут быть включены одновременно.

## Cписок визуализаций (visualizations)

Визуализация это отображение данных на карте. Сейчас доступны Точки, Полигоны, Линии и Тепловая карта.
