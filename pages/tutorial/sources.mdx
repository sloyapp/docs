# Cписок источников данных (sources)

Допустим один из источников данных, который нам понадобится для визуализаций это geojson-файл. Тогда его описать можно так:

```json
{
  "sources": [
    {
      "id": "source-id",
      "path": "/points.geojson",
      "type": "geojson"
    }
  ]
}
```
