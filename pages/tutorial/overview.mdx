# Начало работы

Это технический раздел. Если вы не хотите разворачивать Слой на своём сервере самостоятельно, напишите нам. Мы вам поможем.

## Установка приложения

1. Проверьте что на компьютере установлен [node.js](https://nodejs.org/en/download/package-manager) версии 18+
2. Скачайте шаблон из [https://github.com/sloyio/sloy-template](https://github.com/sloyio/sloy-template) (приватный репозиторий, доступ по запросу)

В шаблоне собрано множество примеров по аналогии с которыми можно собрать свой портал.

## Запуск и остановка приложения.

1. Установить зависимости — `npm i`
2. Запустить проект в режиме разработки — `npm run dev`
3. Собрать публичный проект можно командой `npm run build`. Папку dist достаточно выложить на хостинг.

## Удаление приложения

Достаточно удалить файлы приложения.

## Решение типовых проблем

Версия nodejs должна быть старше 18
