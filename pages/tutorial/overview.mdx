# Руководство пользователя

Геопортал описывается в виде конфигурации в формате JSON. Конфигурация включает в себя настройку:

- источников данных
- слоёв
- визуализаций
- карточек
- фильтров
- копирайтов

[Пример конфигурации слоёв в папке /src/gorodets [архива с примером](http://static.anycraft.tech/map/sloy-template.zip)

[Результат](https://sloy.io/gorodets/)

## Cписок источников данных (sources)

Поверх базовой карты можно накладывать собственные данные в различных форматах, например, в geojson, mvt\pbf или картинках. Ваши данные также могут храниться в postgis базе данных.

[Подробнее про API](/tutorial/sources)

## Cписок слоёв (layers)

Слои отображаются в левой части странице. Например, «Количество жителей», «Здания», «Возраст зданий», «Коммерция», «Туристические маршруты» на скриншоте ниже. Все слои могут быть включены одновременно.

![vis-layers](/vis-layers.png)

[Подробнее про API](/tutorial/layers)

## Cписок визуализаций (visualizations)

Визуализация это отображение данных на карте. Сейчас доступны Точки, Полигоны, Линии и Тепловая карта.

| Формат                              | Пример                                           |
| ----------------------------------- | ------------------------------------------------ |
| Точки![vis-points](/vis-points.png) | Полигоны ![vis-fill](/vis-fill.png)              |
| Линии![vis-lines](/vis-lines.png)   | Тепловая карта![vis-heatmaps](/vis-heatmaps.png) |

[Подробнее про API](/tutorial/visualizations)
