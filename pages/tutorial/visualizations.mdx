# Cписок визуализаций (visualizations)

Визуализация это отображение данных на карте.

Сейчас доступны: Точки, Полигоны, Линии и Тепловая карта.

## Примеры:

### Точки

![vis-points](/vis-points.png)

```json
{
  "layers": [
    {
      "id": "layer-id",
      "title": "Коммерция",
      "visualizations": [
        {
          "id": "vis-id",
          "mapLayerProps": {
            "type": "circle",
            "paint": {
              "circle-color": "#93fa16",
              "circle-stroke-width": 1,
              "circle-radius": 5
            }
          }
        }
      ]
    }
  ]
}
```

### Полигоны

![vis-fill](/vis-fill.png)

```json
{
  "layers": [
    {
      "id": "layer-id",
      "title": "Здания",
      "visualizations": [
        {
          "id": "vis-id",
          "mapLayerProps": {
            "type": "fill",
            "paint": {
              "fill-color": "#ff0062"
            }
          }
        }
      ]
    }
  ]
}
```

### Линии

![vis-lines](/vis-lines.png)

```json
{
  "layers": [
    {
      "id": "layer-id",
      "title": "Туристические маршруты",
      "visualizations": [
        {
          "id": "vid-id",
          "source": "source-id",
          "type": "map",
          "mapLayerProps": {
            "type": "line",
            "paint": {
              "line-width": 2
            }
          }
        }
      ]
    }
  ]
}
```
