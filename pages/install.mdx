# Начало работы

Слой является коробочный веб-приложением, которое разворачивается на сервере клиента. Достаточно самого базового хостинга или сервера. Для того чтобы развернуть систему у себя на сервере необходимо иметь базовые знания языка JavaScript.

## Подготовка к установке

1. Проверьте что на компьютере установлен язык программирования node.js версии 18+. [Инструкция с официального сайта](https://nodejs.org/en/download/package-manager).
2. Скачайте [архив шаблона](http://static.anycraft.tech/map/sloy-template.zip "download")

В шаблоне собрано множество примеров по аналогии с которыми можно собрать свой портал.

Поддерживаются большинство браузеров старше 2012 года (те что имеют поддержку webgl)

## Запуск и остановка приложения.

1. Установить зависимости — `npm i`
2. Запустить проект в режиме разработки — `npm run dev`
3. Собрать публичный проект можно командой `npm run build`. Папку dist достаточно выложить на хостинг.

## Удаление приложения

Достаточно удалить файлы приложения.

## Решение типовых проблем

Версия nodejs должна быть старше 18
