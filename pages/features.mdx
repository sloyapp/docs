# Функциональные характеристики

![Карта Екатеринбурга](/map_ekb.png)

Функциональные возможности программы:

- позволяет визуализировать очень большое количество данных (сопоставимо минимум с 300+ мегабайт geojson одновременно);
- настраивать внешний вид визуализаций и фильтров с помощью конфига;
- отображать несколько слоев одновременно;
- собирать данные с других сервисов через их API;
- отображать 3D-модели зданий;
- настраивать и оформлять картографическую подоснову;
- переключать языки интерфейса;
- переключать темы интерфейса;
- управлять копирайтами и метаданными;
- подключать данные в разных географических проекциях (EPSG:32638, EPSG:28408 и т. д.).
- Использовать низкоуровневое API;
- поддержка векторных тайлов и PMTiles.

В конфигурации можно указать источники данных, стиль оформления и вспомогательные фильтры. [Подробнее](/tutorial/overview)
