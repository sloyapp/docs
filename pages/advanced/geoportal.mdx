# Создание геопорталов

Пошаговая инструкция:

1. Проверить что на компьютере установлен [node.js](https://nodejs.org/en/download/package-manager) версии 18+
2. Скачать шаблон из [https://github.com/sloyio/sloy-template](https://github.com/sloyio/sloy-template) (приватный репозиторий, доступ по запросу)
3. Установить зависимости `npm i`
4. Запустить проект через `npm run dev`
5. Проект статичный, достаточно его собрать через `npm run build` и выложить на хостинг

В примере собрано множество источников в формате geojson

Если у вас есть свои базовые данные, обратитесь к нам, мы поможем.
